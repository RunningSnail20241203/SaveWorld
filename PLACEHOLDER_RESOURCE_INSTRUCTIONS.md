# 占位资源生成说明

## 概述
我已经为您创建了占位资源生成器，可以自动生成所有缺失的UI资源。

## 使用步骤

### 1. 打开Unity项目
Unity应该已经启动并打开了项目。如果没有，请手动运行：
```
D:\Work\release_mobile\P4\unity_2022\WindowsEditor\x64\Release\Unity.exe -projectPath "d:/Test/SaveWorld"
```

### 2. 运行占位资源生成器
在Unity编辑器中：

1. **打开菜单**: 点击顶部菜单栏的 `Tools` → `生成占位资源`
2. **选择生成方式**:
   - **一键生成所有占位资源** - 推荐，生成所有缺失资源
   - **单独生成** - 可以分别生成不同类型的资源

### 3. 等待生成完成
生成过程可能需要1-3分钟，请耐心等待。

### 4. 验证资源
生成完成后，检查以下目录：
- `Assets/Resources/Icons/Items/` - 物品图标
- `Assets/Resources/Icons/UI/` - UI图标
- `Assets/Resources/Audio/` - 音频文件
- `Assets/Resources/WeChat/` - 微信相关资源

3. **资源替换**: 生成真正的美术资源后，直接替换同名文件即可

## 故障排除

### 如果菜单没有显示
1. 确保脚本编译完成
2. 检查Unity控制台是否有错误
3. 重新启动Unity编辑器

### 如果生成失败
1. 检查目录权限
2. 确保磁盘空间充足
3. 查看Unity控制台错误信息
